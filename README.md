
# **Installation**

## **install gcc:**

### **Arch Linux**

```sudo pacman -S gcc```

### **Debian**

```sudo apt-get install gcc```

### **Mac**

```brew install gcc```

### **Windows**

Install from the [sourceforge](https://sourceforge.net/projects/gcc-win64/) website

## **clone & compile:**

```git clone https://github.com/jayuck/deadc```

```cd deadc```

```gcc main.c -o deadc```

## **use:**

```deadc <path_to_deadfish_file>```
